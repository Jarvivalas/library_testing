# COMP.SE.200 Group Project: E-commerce Utility Library Testing
This is for a uni course, so not intended for other use.
## Getting Started
1. Make sure you have **Node.js** installed locally (recommended v.14 or later).
2. Clone the repo
3. Run `npm install`
4. Verify that everything is working correctly by running tests `npm test` and coverage `npm run coverage`.
